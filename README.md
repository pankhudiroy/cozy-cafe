# Cozy Cafe Website

A clean, responsive, and elegant landing page designed for a local boutique cafe. This project focuses on a modern UI, mobile-first design, and seamless user experience.

## 🚀 Live Demo
(https://illustrious-banoffee-2e2da3.netlify.app/)

## 🛠️ Tech Stack
* **HTML5** - Semantic structure.
* **CSS3** - Responsive design, Flexbox, and custom animations.
* **JavaScript** - Mobile navigation menu functionality.
* **Font Awesome** - Social media icons.
* **Google Fonts** - Elegant typography (Great Vibes).

## ✨ Key Features
* **Fully Responsive:** Optimized for desktop, tablet, and mobile devices.
* **Mobile-First Navigation:** Features a slide-out drawer menu for smaller screens.
* **Elegant Branding:** Custom typography and a cohesive color palette.
* **Hero Section:** High-impact visual section with clear call-to-action (CTA).
